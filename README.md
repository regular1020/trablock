# trablock

## 개요

여행 플래너 & 여행 가계부

여행 계획들을 블럭으로 만들고 배치해서 여행 플랜을 완성할 수 있습니다.

## 사용법

### 회원가입

현재 구글 계정을 통한 로그인을 제공하고 있습니다.

### 새 여행 생성

메인 페이지 상단 메뉴버튼 클릭 후 새 여행 선택

시작시간, 메모 선택 입력
